Status Code :
200 - Success:
200 - ok 
201 - created 
204 - No content (Like data deleted)

400 - client mistake
400 - Bad request
401 - Authorization (Invalid credentials)
403 - Forbidden (Authentication succeeded but authenticated user has no access)
404 - Not Found (Request resource doesnt exist)
405 - Method Not Allowed
409 - conflict (Conflict between two simultaneous updates)

500 - server error
500 - internal server error (Server did a mistake client cant do anything)
